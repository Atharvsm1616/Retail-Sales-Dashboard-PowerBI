
# 📊 Retail Sales Dashboard using Power BI

This project is an interactive Retail Sales Dashboard built using Power BI to analyze retail transaction data and generate business insights.

---

## 🔍 Project Overview

This dashboard helps in analyzing:
- Sales performance
- Customer segmentation
- Product performance
- Geographic distribution of sales

---

## 📈 Key Insights

- Majority of sales come from a few countries
- Sales show strong growth in later months
- A small number of products contribute to most of the revenue

---

## 📊 Dashboard Features

- KPI Cards (Total Sales, Customers, Orders)
- Sales Trend Analysis
- Product Performance Analysis
- Customer Segmentation (Donut Chart)
- Geographic Insights
- Gauge Chart for performance tracking

---

## 🛠 Tools Used

- Power BI
- Power Query
- DAX (Data Analysis Expressions)
